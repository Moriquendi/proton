BackgroundStyle

public struct BackgroundStyle

Additional style for background color attribute. Adding BackgroundStyle attribute in addition to backgroundColor attribute will apply shadow and rounded corners as specified.

Note

This attribute had no effect in absence of backgroundColor attribute.
